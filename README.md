# Resume Ranking System

A web-based Resume Ranking System built using Python, Flask, HTML, CSS, and SQLite.

## Features

- User Login System
- Upload Job Description
- Upload Multiple Resumes
- Resume Ranking Based on Skill Matching
- Candidate Score Calculation
- Results Dashboard
- Logout Functionality

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite

## Project Workflow

1. User Login
2. Upload Job Description
3. Upload Resumes
4. Skill Matching
5. Candidate Ranking
6. Display Ranked Results
